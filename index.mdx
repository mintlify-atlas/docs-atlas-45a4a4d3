Levi.dev is a full-stack Next.js application that combines a personal portfolio with a complete digital product storefront. It features a cyberpunk/matrix aesthetic with animated 3D backgrounds, integrated Stripe and PayPal payments, license key management, and a comprehensive admin dashboard — all in a single deployable application.

## Get started

<Steps>
  <Step title="Clone and install">
    Clone the repository and install dependencies with your package manager.
  </Step>
  <Step title="Configure environment">
    Set up your `.env` file with database, auth, and payment provider credentials.
  </Step>
  <Step title="Set up the database">
    Run Prisma migrations and optionally seed initial data.
  </Step>
  <Step title="Run the app">
    Start the development server and visit `http://localhost:3000`.
  </Step>
</Steps>

<Note>
  See the [Quick Start guide](/quickstart) for detailed instructions on each step.
</Note>
